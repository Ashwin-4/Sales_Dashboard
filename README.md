# Sales_Dashboard
**Power BI Dashboard**
 **Project** **Objective**:To contribute to the success of a business by utilizing data analysis techniques,specifically focusing on time series analysis to provide valuable insights and accurate sales forecasting
 **Dataset:** Financial dataset Imported through Excel database. 
 ****Insights**:** Total Orders is 22K,Total Sales is 1.57M,total Profit 175.26M,Average ship days is 4,Sales By Month,Region and many more as shown in project. 
 **Steps**: i)Imported Dataset using Excel database 
        ii)Cleaning the Data by using Power BI(using excel Functions i.e remove blank columns replace N/A to 0,checking Datatype is correct or no etc) 
        iii)Written DAX Queries for easy calculation of fields and use it in our project. 
        iv)Created Dashboard from scratch (using clustered bar Charts ,Line chart for forecast ,slicer etc) 
        v)Export to PDF and uploaded in Github.
